# Votazioni Poesie - Notte del Classico

Questo repository contiene il sito web per la **Notte Nazionale del Liceo Classico**, dove gli utenti possono votare le poesie tramite un QR code o un link diretto.

## Come funziona il sito

- Mostra un **QR code** che porta a un modulo Google Forms per la votazione.
- Include un pulsante “Vota Ora” che apre il modulo.
- Riproduce un **audio di sottofondo** che parte al primo clic del body o del pulsante.
- Design responsive e moderno, adatto a desktop e mobile.
